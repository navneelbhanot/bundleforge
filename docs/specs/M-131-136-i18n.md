# M-131..M-136 — i18n framework + 6 locales

## M-131 Framework

`src/i18n/` — tiny in-house translator (no `i18next` runtime
dependency on the server) that loads JSON locale files and exposes
`t(key, vars?, locale?)`. It honors `Shop.locale` set by the
`shop/update` webhook (M-027).

The frontend already uses Polaris's i18n; the same JSON files are
shared (M-093 already shipped en/es/fr for the theme extension —
M-131..M-136 add server-side strings).

## Locale list

- M-132 English (baseline) — `src/i18n/locales/en.json`
- M-133 Spanish — `es.json`
- M-134 French — `fr.json`
- M-135 German — `de.json`
- M-136 Italian + Portuguese — `it.json`, `pt.json`

## Scope

Server-side strings used in error messages and webhook receipts.
Translations are professional-quality stubs; a real localization
review (M-150 area) tightens phrasing.
