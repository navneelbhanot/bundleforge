# BundleForge — Merchant Help

Practical, no-fluff documentation for merchants using BundleForge.

> **Not finding your question?** [docs/help/troubleshooting.md](troubleshooting.md)
> covers the most common issues. If that doesn't help, contact support
> via the address in your App Store listing.

## Start here

- [**Getting started**](getting-started.md) — install the app, create
  your first bundle, see it on your storefront. ~5 minutes.

- [**Bundle types**](bundle-types.md) — what each of the 13 bundle
  types does, when to use it, and how to configure it.
- [**Pricing rules**](pricing.md) — fixed / percentage / tiered /
  volume / BOGO discounts, how rules stack, and how to verify cart
  pricing matches checkout.
- [**Inventory**](inventory.md) — how bundle sales affect component
  stock, the audit log, and what happens when a component runs low.
- [**Storefront integration**](storefront.md) — where bundles render,
  the five theme app blocks, and the App Proxy URL pattern.
